Game

How to play

Custom version of the game "jetpack joyride" in which the player is a plane with customizable characteristics that flies avoiding different types of obstacles (air, ground, meteorological ...), in addition the difficulty is added that the plane loses fuel with the distance, therefore; the player must control that it does not run out while doing the course simultaneously.

Objectives

Travel the maximum distance in the shortest possible time.

At the same time he tries to get all the coins he finds on his way.
